# Little Brain Games

A learning game for children ages 5-7 with four mini-games that strengthen memory, pattern recognition, puzzle-solving, and logical thinking.

## What's here

- `kids-game/index.html` — the game itself, a single self-contained HTML file. Open it in any modern browser to play.
- `kids-game/index_v3.html` — earlier archive copy of the game.
- `docs/superpowers/specs/` — design specs:
  - `2026-04-24-kids-learning-game-design.md` — original game design
  - `2026-04-25-little-brain-games-android-phase1-design.md` — design for the Android app (Phase 1)

## How to play

Double-click `kids-game/index.html`, or open it from your browser. No install, no internet required.

## Games

- 🧠 **Memory** — match pairs of pictures, or pictures with their words
- 🎨 **Pattern** — figure out what comes next in a sequence (shapes, letters, numbers, or words)
- 🧩 **Puzzle** — drag jigsaw pieces onto the board (10/25/50/75/100 pieces). Built-in scenes plus an upload-your-own-picture option.
- 🦉 **Logic** — pick the odd one out
